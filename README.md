# UniApply

A study-abroad application platform built for South and Southeast Asian students — starting with Nepal. UniApply replaces expensive consultancies (NPR 60,000+) with an AI-powered toolkit that guides students through every step of the graduate school application process.

**Live:** [uandiapply.com](https://www.uandiapply.com/)

---

## What it does

### University Management
- Add and track universities across Dream, Target, and Safety categories
- AI-powered university suggestions based on your profile, GPA, test scores, and target countries
- Fit score badges (Strong Fit / Borderline / Reach) comparing your profile against program requirements
- Application deadline tracking with email reminders at 30, 14, and 7 days out
- Funding awareness — PhD programs flagged as typically funded (TA/RA)

### Application Checklist
- Per-university checklist tracking SOP, LORs, transcript, test scores, financial docs, and resume
- Enriched with AI-fetched program requirements (GRE/TOEFL minimums, credential evaluation, application fee)
- Progress bar always visible in the university header — click to jump to the full checklist

### Statement of Purpose (SOP)
- Multi-draft workspace with a rich text editor (TipTap)
- AI draft generator — produces a structured 500–700 word SOP from your profile
- AI critique with scores across four dimensions: clarity, specificity, fit, and narrative
- Critique score history with delta tracking across multiple runs
- Shareable review links for advisor or peer feedback (30-day expiry)

### Documents & LORs
- Upload and tag documents (transcripts, test scores, financials, resume) to specific universities
- View and download tagged documents directly from each university's page
- Letter of Recommendation tracker — manage recommenders, deadlines, and submission status per university
- AI-generated LOR request email drafts

### Profile
- GPA entry with scale normalization (US 4.0, CGPA 10, percentage)
- Test scores (GRE, TOEFL, IELTS, GMAT, SAT, ACT)
- Academic background, research interests, work experience, extracurriculars, community service
- Target countries and career goals used to personalize AI suggestions

### Onboarding
- 5-step wizard: degree level → academic profile → goals + country preferences → experience → AI university suggestions
- Dream/Target/Safety suggestions generated by Claude on first run, selectable and saved in one click

### Timeline Calculator
- Backwards milestone calculator from your enrollment date
- 14 milestones for Masters/PhD covering every major application step
- Dashboard widget showing your next 3 upcoming milestones

## Running locally

**Prerequisites:** Node.js 18+, an Anthropic API key, a College Scorecard API key (free at [api.data.gov](https://api.data.gov/signup/))

```bash
# Clone
git clone https://github.com/simantstha/uniapply.git
cd uniapply

# Backend
cd server
cp .env.example .env        # fill in your keys
npm install
npx prisma migrate dev
npm run dev                 # runs on :3001

# Frontend (new terminal)
cd client
npm install
npm run dev                 # runs on :5173
```

**Required environment variables (`server/.env`):**

```
DATABASE_URL=file:./dev.db
JWT_SECRET=your-secret
ANTHROPIC_API_KEY=sk-ant-...
COLLEGE_SCORECARD_API_KEY=your-key
CLIENT_URL=http://localhost:5173
```

## Deployment

- **Backend** — Fly.io with a 1GB persistent volume for SQLite
- **Frontend** — Vercel, zero-config
- **CI/CD** — GitHub Actions deploys frontend to Vercel and backend to Fly.io on push to `main`, only when the respective directory has changed

## Why

University application consultancies in Nepal charge NPR 50,000–80,000 for services that are largely template-based. UniApply aims to give every student access to the same quality of guidance at a fraction of the cost — or free.
